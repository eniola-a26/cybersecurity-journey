# Introduction to Cyber Security

## 🧠 Objective
Understand the basic concepts of cybersecurity, including offensive and defensive security, and explore career paths in the field.

---

## 📚 Key Concepts

### 🔴 Offensive Security
- Focuses on attacking systems to find vulnerabilities
- Uses tools and techniques like penetration testing
- Goal: Identify weaknesses before attackers do

### 🔵 Defensive Security
- Focuses on protecting systems and data
- Includes monitoring, detection, and response
- Goal: Prevent and respond to cyber attacks

### 🧑‍💻 Careers in Cyber
- Security Analyst
- Penetration Tester
- Security Engineer
- Incident Responder
