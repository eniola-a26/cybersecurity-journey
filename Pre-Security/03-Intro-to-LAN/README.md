# Intro to LAN

## 🧠 Objective
Understand how local networks (LANs) are structured and how devices communicate within them.

---

## 📚 Key Concepts

### 🌐 LAN (Local Area Network)
- A network covering a small area (home, office)

### 🔷 Network Topologies
- Star → devices connect to central hub
- Bus → single backbone cable
- Ring → devices connected in loop

### 🔢 Subnetting
- Dividing a network into smaller parts
- Improves efficiency and security

### 🔁 ARP (Address Resolution Protocol)
- Maps IP address → MAC address

### 📡 DHCP
- Automatically assigns IP addresses to devices
