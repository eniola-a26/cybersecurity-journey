# OSI Model

## 🧠 Objective
Understand the OSI (Open Systems Interconnection) model and how data moves through network layers.

---

## 📚 Key Concepts

### 🌐 What is the OSI Model?
- A framework that describes how data travels across a network
- Divided into 7 layers

---

## 🧱 OSI Layers (Top → Bottom)

### 7. Application
- User interaction layer
- Examples: HTTP, FTP

### 6. Presentation
- Data formatting and encryption
- Ensures data is readable

### 5. Session
- Manages connections between devices

### 4. Transport
- Ensures reliable data transfer
- Example: TCP

### 3. Network
- Handles routing and IP addresses

### 2. Data Link
- Transfers data between devices on the same network

### 1. Physical
- Physical transmission of data (cables, signals)

---


## 🔄 Data Flow (Encapsulation)

- Data travels from Layer 7 → Layer 1 when being sent
- Each layer adds information to the data
- This process is called encapsulation

- When data is received:
  - It travels from Layer 1 → Layer 7
  - Each layer removes its information

---

## 📦 Data Names by Layer
- Layer 4 → Segment (TCP) / Datagram (UDP)
- Layer 3 → Packet
- Layer 2 → Frame
