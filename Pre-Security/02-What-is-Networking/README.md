# What is Networking?

## 🧠 Objective
Understand the basics of networking, including how devices communicate and key networking concepts.

---

## 📚 Key Concepts

### 🌐 What is a Network?
- A network is a collection of devices connected together to share data
- Examples: Internet, home Wi-Fi, office networks

### 🖥️ Devices on a Network
- Computers, servers, routers, switches
- Each device has an identity (IP address)

### 📡 IP Address
- A unique identifier for a device on a network
- Example: 192.168.1.1


### 🔄 How Communication Works
- Devices send and receive data using protocols
- Data is broken into packets and transmitted across the network
